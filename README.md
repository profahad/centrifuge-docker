## Centrifuge Docker

Centrifugo is a scalable real-time messaging server in language-agnostic way. Set up once and forever.

### Server Installation

- `install docker`
- `$ git clone https://github.com/profahad/centrifuge-docker`
- `$ cd centrifuge-docker`
- `$ docker-compose build`
- `$ docker-compose up`
